# 기둥 축력 계산서 (Column Axial Load Calculator)

분담면적과 단위면적당 하중(DL·LL)만 입력하면 기둥별 **사용하중 P = D+L** 과 **계수하중 Pu = 1.2D+1.6L** 을 즉시 산정하는 예비설계용 웹 도구입니다. 다층 건물의 층별 누적 축력 계산을 지원하며, 서버·빌드 없이 `index.html` 파일 하나로 동작합니다.

## 주요 기능

- **하나의 기둥 목록, 두 개의 화면** — 기둥은 층별 상세계산에서 정의하며, 기둥축력 일람표는 그 목록을 그대로 반영해 각 기둥의 최하층(1층 기둥, 기초 전달) 축력을 자동 집계
- **층마다 다른 하중 지원** — 슬래브 레벨명과 층별 하중(DL·LL)은 **층의 속성**으로 한 번만 입력하면 모든 기둥에 공통 적용(표에서 옅은 파랑 셀). 예: 로비층 활하중 상향, 기계실층 고정하중 상향 등을 층 단위로 반영. 분담면적(Lx·Ly)만 기둥별로 입력
- **아래층(1F·지하층) 추가** — 지상 층수 N(지붕 포함)은 숫자로 입력하고, 1F 이하 층은 [＋ 아래층 추가] 버튼으로 1F → B1 → B2 … 순서로 붙임(새 층에는 바로 위 층의 하중이 복사됨). 1F 슬래브 → B1 기둥, B(k) 슬래브 → B(k+1) 기둥에 재하되며, 최하층 바닥은 온통기초·토간 슬래브로 보아 기둥 재하에서 제외. 기둥 명칭(…2층, 1층, B1층, B2층 기둥)과 기초 전달 표시가 자동 부여되고, 삭제한 아래층은 재추가 시 입력값이 복원됨
- **미지지 층 처리** — 기둥이 지지하지 않는 층은 면적을 0으로 입력(예: 저층부 기둥). 지지층수 n은 면적 > 0인 층의 수로 자동 산정
- **기둥축력 일람표** — 기둥별 대표 치수(지지층 동일 시)·분담면적(상이 시 범위 표시)·지지층수 n·ΣD·ΣL·P·Pu를 요약. 합계행은 전체 중력하중(기초설계 참고), 미입력이 있는 기둥은 붉은색으로 표시되고 합계에서 제외. 행의 [상세] 버튼으로 해당 기둥의 층별 상세계산으로 즉시 이동
- **층별 상세계산 (다중 기둥)** — 기둥부호를 칩(chip) 형태로 추가·선택해 기둥마다 면적 구성을 따로 관리
  - **건물 층수 N을 입력하면 슬래브 레벨이 전 기둥 공통으로 자동 생성·삭제**되고 레벨명(R, NF … 2F)이 다시 매겨짐. 남는 층의 입력값(하중·면적)은 유지되며, 줄였다가 다시 늘리면 직전 입력값이 복원됨
  - 기둥 추가 시 현재 선택된 기둥의 면적 구성이 복사되어 면적만 바꿔 쓰기 편리
  - 각 행의 누적축력 = 해당 레벨 직하부 기둥의 축력, 마지막 행 = 1층 기둥(기초 전달하중, 스탬프로 강조 표시)
- **적용하고 일람표 보기** — 층별 상세계산의 모든 수정은 즉시 자동 저장되어 일람표에 반영되며(하단에 ✓ 반영 시각 표시), [적용하고 일람표 보기] 버튼으로 일람표로 이동해 갱신된 값을 하이라이트로 확인 가능
- **1.4D 지배 자동 경고** — 활하중이 매우 작아 1.4D 조합이 1.2D+1.6L을 상회하면 해당 기둥에 표시
- **PDF로 저장 / 계산서 인쇄** — [PDF로 저장] 버튼으로 일람표·층별 상세를 모두 담은 A4 가로 PDF 파일을 바로 다운로드(다중 페이지 자동 분할). 오프라인 등으로 PDF 모듈을 못 불러오면 [계산서 인쇄] → 'PDF로 저장'으로 동일한 결과를 얻을 수 있음
- **03 NOTES 화면 전용** — 03 NOTES는 화면 참고용으로만 표시되며 PDF·인쇄 출력물에는 포함되지 않음
- **엑셀로 저장** — 기둥축력 일람표와 전체 기둥의 층별 상세계산을 2개 시트로 담은 .xlsx 파일 다운로드

## 산정식

```
분담면적  A  = Lx × Ly
고정하중  D  = α × A × [ DL(지붕층) + (n−1) × DL(기준층) ]
활하중    L  = A × [ LL(지붕층) + (n−1) × LL(기준층) ]
사용하중  P  = D + L                (사용성·기초 지내력 검토용)
계수하중  Pu = 1.2D + 1.6L          (강도설계용)
```

- **지지층수 n**: 해당 기둥이 지지하는 슬래브 레벨 수(지붕층 포함). 예: 5층 건물의 1층 기둥 → n = 5
- **자중할증계수 α**: 기둥 자중 등을 반영해 고정하중 D에만 곱함(통상 1.00~1.10)
- 층별 상세계산: `층별 D = A×DL`, `ΣD = ΣD(상부) + α×D`, `ΣL = ΣL(상부) + L` — DL·LL은 층별 값 사용

설계 조건의 기본값(지붕층 DL 6.5 / LL 1.0, 기준층 DL 7.0 / LL 2.5 kN/m²)은 층수를 늘릴 때 새 층에 채워지는 값(지붕층→R, 기준층→지상층·1F)이며 KDS 41 10 15(건축구조기준 설계하중)를 참고했습니다. 단위: kN, kN/m² (1 kN ≈ 0.102 tonf).

## 사용법

1. `index.html`을 브라우저에서 열기(더블클릭). 별도 설치·인터넷 연결 불필요
2. 상단 표제란에 프로젝트·작성일·작성자 입력
3. **01 설계 조건**에서 지붕층/기준층 DL·LL과 α 입력 — 파란색 값이 입력, 검은색 값이 자동 계산
4. **층별 상세계산** 탭: 지상 층수 N 입력, 필요 시 [＋ 아래층 추가]로 1F·B1… 추가 → 층별 DL·LL 수정(전 기둥 공통) → 칩에서 기둥 선택(＋ 기둥 추가로 늘리기) → 기둥별 면적 입력(미지지 층은 0)
5. [PDF로 저장]으로 계산서 파일 다운로드 또는 [계산서 인쇄]로 출력
6. **기둥축력 일람표** 탭: 정의된 모든 기둥의 축력 요약 확인. 기둥부호·비고 수정, ＋ 기둥 추가, [상세]로 이동, × 로 삭제 가능
6. 「계산서 인쇄」로 출력 또는 PDF 저장

## GitHub Pages로 배포하기

### 방법 A — 웹 브라우저만으로 (git 불필요)

1. GitHub 로그인 → 우측 상단 **+ → New repository** → 이름 `column-axial-load-calc`(예시), **Public** 선택 후 생성
2. 저장소 화면에서 **Add file → Upload files** → `index.html`, `README.md`, `LICENSE`, `.gitignore`를 끌어다 놓고 **Commit changes**
3. **Settings → Pages → Build and deployment**에서
   - Source: `Deploy from a branch`
   - Branch: `main` / `/ (root)` 선택 후 **Save**
4. 1~2분 후 `https://<사용자명>.github.io/column-axial-load-calc/` 에서 접속 가능

### 방법 B — git 명령어로

```bash
# 압축 해제한 폴더에서
cd column-axial-load-calc
git init
git add .
git commit -m "기둥 축력 계산서 v1.0"
git branch -M main
git remote add origin https://github.com/<사용자명>/column-axial-load-calc.git
git push -u origin main
```

이후 방법 A의 3~4단계(Settings → Pages)만 동일하게 설정하면 됩니다. 파일을 수정한 뒤에는 `git add . && git commit -m "수정 내용" && git push`로 갱신하며, Pages는 자동으로 재배포됩니다.

> PDF·엑셀 저장 기능은 cdnjs에서 라이브러리를 불러오므로 온라인 환경에서 동작합니다. 배포 후 실제 브라우저에서 두 버튼을 한 번 확인해 보세요.

## 유의사항

- 지하층의 흙·수압, 지하 외벽 자중 등은 본 도구의 범위 밖이므로 별도 고려해야 합니다
- 활하중 저감은 미적용(안전측)입니다. 영향면적 36 m² 이상 시 KDS 41 10 15 4.7에 따라 별도 저감 가능
- 본 도구는 **예비설계용** 중력 축력 산정 도구이며, 최종 설계는 구조해석 결과로 검증해야 합니다
- 입력값은 브라우저에 저장되지 않습니다. 보관이 필요하면 인쇄 기능으로 PDF 저장을 이용하세요
